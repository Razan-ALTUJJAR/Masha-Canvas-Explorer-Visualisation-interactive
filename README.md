##  Objectif
Explorer une image comme une **donnée interactive** : zoom, drag, mapping couleur (rainbow), rendu temps réel.

## 🛠 Stack

- HTML5 / Canvas API
- JavaScript (vanilla)
- Détection de cycle couleur

##  Aspects Data Science
- Transformation pixel → couleur dynamique
- Algorithme de mapping couleur continu

## Aspects Data Analyst
- Interaction utilisateur (zoom / drag)
- Exploration visuelle d’une donnée (image)

## ✨ Fonctionnalités
-  Image Masha 
-  Arrière‑plan arc‑en‑ciel **dans l’image**
-  Zoom + / – (boutons + molette)
- Déplacement quand zoomée
